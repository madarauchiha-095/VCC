# Event Management System Frontend

React + Vite Frontend for the Institutional Event Resource Management System.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Frontend will run on `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- Role-based authentication (COORDINATOR, HOD, DEAN, HEAD, ADMIN)
- Event creation, submission, and approval workflow
- Resource allocation management
- Venue booking with conflict detection
- Real-time event status tracking
- Admin panel for venue and resource management
